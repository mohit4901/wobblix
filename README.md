# ⚡ WOBBLIX CLOTHING | Premium Gen-Z Streetwear

**Wobblix Clothing** is a high-performance, full-stack E-commerce ecosystem built for the modern streetwear market. It features a brutalist-minimalist "Bone & Red" design, seamless Razorpay integration, and a dedicated admin command center.

## 🚀 Key Features
- **Brutalist UI**: A sharp, high-contrast aesthetic designed for Gen-Z impact.
- **Razorpay Exclusive**: Secure, native Indian payment gateway integration.
- **Fluid Responsiveness**: 100% device-agnostic from 4K TVs to iPhone 13 mini.
- **MERN Stack**: High-performance architecture using MongoDB, Express, React, and Node.js.
- **Admin Command Center**: Complete control over inventory, orders, and tracking.

## 🛠️ Tech Stack
- **Frontend**: React.js, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express.js, JWT Authentication
- **Database**: MongoDB Atlas
- **Storage**: Cloudinary (Image Assets)
- **Payments**: Razorpay

## 📦 Installation

1. **Clone the Repo**
```bash
git clone https://github.com/mohitmudgil/wobblix.git
cd wobblix
```

2. **Environment Setup**
Create `.env` files in `backend`, `frontend`, and `admin` as per the provided templates.

3. **Install Dependencies**
```bash
# In each folder
npm install
```

4. **Run Development**
```bash
# Backend
npm run server

# Frontend / Admin
npm run dev
```

## 🏗️ Architecture
- `/frontend`: The main storefront.
- `/admin`: The brand management dashboard.
- `/backend`: The central API engine.
